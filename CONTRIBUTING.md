# Contributing

Thanks for helping build the reference list for Android App Functions apps.

## Before You Submit

- Search existing issues and pull requests to avoid duplicates.
- Make sure the app is real and publicly identifiable.
- Include at least one verifiable source for App Functions support.
- Keep descriptions concise and factual.

## Entry Format

Add or update rows using this format:

| App | Developer | Play Store | Exposed Functions (if known) | Evidence |
| --- | --- | --- | --- | --- |
| Example App | Example Dev | https://play.google.com/... | createNote, searchNotes | https://developer.android.com/reference/android/app/appfunctions/package-summary |

## Evidence Guidelines

Accepted evidence can include:

- Official Android or vendor documentation.
- Official release notes or changelogs.
- Product demos from official channels.
- Technical analysis with reproducible steps.

Avoid unverifiable claims.

## Pull Request Checklist

- [ ] I verified this app is not already listed.
- [ ] I added at least one verifiable evidence source.
- [ ] I used the required table format.
- [ ] I kept wording neutral and objective.
- [ ] I checked links are valid.

## Review Policy

Maintainers may request edits before merging to keep consistency and quality.

## Scope

This repository lists apps and evidence.
It does not host proprietary code, APKs, or copyrighted assets.
